Vitriolic

University UE4, PC UI and Gameplay Programming

Developed in my final year group project module, Vitriolic is a Top Down Hack'n'Slash singleplayer game set in a Cyberpunk world. Myself, along with 10 other students from both the Computer Games Programming and Computer Games Modelling and Animation courses, designed and developed the game completely from scratch.

Read more...

The game itself was developed for PC in Unreal Engine 4, with a mixture of C++ and UE4 Blueprints. We managed to win the **'Playground Games Best Game Award'** and the **'Best Art Direction'** awards at the Games@Derby Expo 2018.
